KunaQuest

Kuna

KunaQuest es una aplicación educativa móvil diseñada para enseñar el idioma aymara de Bolivia mediante trivias interactivas, permitiendo que los usuarios aprendan a través de preguntas de opción múltiple, verdadero/falso y actividades de enlazado de palabras.

Meta
Final

La app busca incentivar el aprendizaje del aymara de manera divertida y accesible, preservando la cultura aymara y ayudando a que las nuevas generaciones se familiaricen con el idioma y sus tradiciones​.

Vista
Gamer

El juego presenta preguntas en formato de trivia, con mecánicas de gamificación como puntos, niveles y recompensas, que motivan a los usuarios. Además, ofrece retroalimentación inmediata, permitiendo que los usuarios evalúen su progreso en tiempo real​.

¿Por
Que?

El aymara es parte importante de la identidad cultural en Bolivia. El proyecto busca contribuir a la preservación de esta lengua originaria, que está en riesgo de desinterés entre las nuevas generaciones, promoviendo su aprendizaje y uso cotidiano​.

Se
Uso

Para desarrollar KunaQuest, se usaron herramientas como Android Studio para la programación y Firebase para la autenticación y almacenamiento. También se utilizaron bases de datos SQL Lite y lenguajes como Java y Kotlin.
